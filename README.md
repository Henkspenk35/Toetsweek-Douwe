# Toetsweek Douwe

Deze versie bevat een gecorrigeerde index:
- Frans U4–5 verwijst naar `frans_u45.html`
- daarna volgen Engels zinnen, Engels phrases, grammatica, Latijn, Nederlands en Biologie
- de oude dubbele tegel voor Frans is verwijderd

## Publiceren
Upload alle bestanden in deze map naar de root van je repository en overschrijf bestaande bestanden.
Daarna in GitHub:
Settings → Pages → Deploy from a branch → main → / (root) → Save

## Belangrijk na upload
Doe daarna één harde refresh van de site, zodat de nieuwe service worker en index worden geladen.
